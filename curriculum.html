Curriculum

Overview

  • Create an accessible, responsive, static web page using semantic HTML, which is styled to specification with CSS.
  • Solve problems by writing JavaScript programs.
  • Work with popular Javascript libraries such as jQuery.
  • Build a well-architected, interactive web app which achieves a specified set of user stories.

Week 1

  • Learn the fundamentals of markup coding language HTML.
  • Start building personal website.

Week 2

  • Learn the fundamentals of markup coding language CSS.
  • Add style to personal website.

Week 3

  • Go deeper in HTML & CSS.
  • Learn how to implement responsive design in your websites.
  • Introduction to flexbox and media quieries.

Week 4

  • Start working with essential coding platforms such as Github and the command line.
  • Create first git repository and deploy personal website through terminal.

Week 5

  • Pick topic for personal project and create HTML & CSS static website.

Week 6

  • Begin learning fundamentals of Javascript.
  • Master variables and functions.

Week 7

  • Get more advanced in Javascript.
  • Practice working with loops and conditional statements.

Week 8

  • Code-along Javascript project.
  • Gain some experience with vanilla JS.

Week 9

  • Explore the jQuery library and complete mini practice projects.

Week 10

  • Practice using jQuery.
  • Complete shopping list application.

Week 11

  • Start working on your own jQuery project.

Week 12

  • Talk about options, industry, and what you can accomplish now:)
