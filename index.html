She Hacks Program Details:

She Hacks is an organization dedicated to teaching women front end web development in the most affordable and effective way possible. We teach you how to use programming as a tool to accomplish what you care about - not a foregin language you will never find relevant to your daily life. Throughout the program, you will create many websites and learn how to advertise to audiences. By the end, you will have your own portfolio to display your creations on. This program is totally free and real-life application based. Fill out the registration form to join us for the first lesson November 5th, 2020.

Curriculum

  • Create an accessible, responsive, static web page using semantic HTML, which is styled to specification with CSS.
  • Solve problems by writing JavaScript programs.
  • Build a well-architected, interactive web app which achieves a specified set of user stories.
  • Work with popular Javascript libraries such as jQuery.
  • Design and build an original client-side web app that integrates with a 3rd-party API.
